#### Клонирование проекта

```console
git clone --recursive https://github.com/timcogames/sway.module_math.git
```

#### Сборка проекта

```console
mkdir build && cd ./build
cmake -DCMAKE_BUILD_TYPE=Release ../
make
```

#### Опции сборки

* `-DCMAKE_BUILD_TYPE=Debug`
* `-DBUILD_TESTS=ON`
